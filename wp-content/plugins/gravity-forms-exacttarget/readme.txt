### Integrate Gravity Forms with ExactTarget

If you use ExactTarget and WordPress, this plugin is for you. Sync Gravity Forms forms with your ExactTarget account - if a linked Gravity Form is submitted, all data will be seamlessly added to ExactTarget.

* Works with ExactTarget accounts with thousands of lists
* Link multiple forms with ExactTarget lists
* Choose to link a Gravity Form with one or more ExactTarget lists (submit once, get added to multiple lists!)
* Uses the ExactTarget XML API
* Includes administrator debugging options

###The Best ExactTarget WordPress Plugin
Using Gravity Forms integration allows you to take advantage of Gravity Forms features, including having a widget subscription box linked with ExactTarget. If you need an <strong>ExactTarget widget</strong>, this plugin is for you.

== Installation == 

1. Upload plugin files to your plugins folder, or install using WordPress' built-in Add New Plugin installer
1. Activate the plugin
1. Go to the plugin settings page (under Forms > Settings > ExactTarget)
1. Enter the information requested by the plugin.
1. Click Save Settings.
1. If the settings are correct, it will say so.
1. Follow on-screen instructions for integrating with ExactTarget.

== Frequently Asked Questions == 

= Does this plugin require Gravity Forms? =
This plugin requires the brilliant [Gravity Forms plugin](http://wordpressformplugin.com/?r=et). __Don't use Gravity Forms? [Buy the plugin](http://wordpressformplugin.com/?r=et)__ and start using this add-on plugin!

= Does this plugin require ExactTarget? =
Well of course it does.

= The plugin doesn't work for me. Why not!? =
If you are using ExactTarget Enterprise 2.0, this plugin won't work for you. It requires the XML API, which Enterprise 2.0 does not support.
